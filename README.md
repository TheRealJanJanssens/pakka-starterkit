## Install

```
cp .env.example .env

fill .env

composer install
npm install

php artisan key:generate 
php artisan pakka-install
php artisan migrate
php artisan db:seed
```

## Bug Fixing
```
php artisan config:clear
php artisan cache:clear
php artisan view:clear
php artisan route:clear
composer dump-autoload
```
